# Timothy Kim Portfolio Site

A static personal portfolio site for showcasing data science, machine learning, sports analytics, and web demo projects.

## Customize

Edit these placeholders before publishing:

- `YOUR_GITHUB_USERNAME`
- `YOUR_LINKEDIN`
- Each project `demoUrl` in `script.js`
- Each project `codeUrl` in `script.js`

## Publish With GitHub Pages

For a personal GitHub Pages site:

1. Create a public GitHub repository named exactly `YOUR_GITHUB_USERNAME.github.io`.
2. Upload `index.html`, `style.css`, `script.js`, and this `README.md` to the repository.
3. Go to repository `Settings` -> `Pages`.
4. Set the publishing source to the `main` branch and root folder.
5. Your site should appear at `https://YOUR_GITHUB_USERNAME.github.io`.

## Project Structure

```text
index.html
style.css
script.js
README.md
```

## Notes

The site uses only HTML, CSS, and JavaScript, so it does not need a build step or package install.
